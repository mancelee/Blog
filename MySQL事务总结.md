# MySQL 事务

首先明确几个概念，
### 一、事务的4个基本要素：ACID
1. A(Atomicity) 原子性：一个事务要么成功要么失败，不存在中间状态。
2. C(Consistency) 一致性： 事务开始前和结束后，数据库的完整性约束没有被破坏。
3. I(Isolation)隔离性: 同一时刻，只允许一个事务请求同一个数据，不同事务之间彼此没有任何干扰。
4. D(Durability)持久性： 事务完成后，事务对数据库的所有更新保存到数据库中，不能回滚。
